Technical Specifications

This is the master document for my PhD Thesis, investigating the issues that may arise from structural breaks, in the context of large macroeconomic factor models.

This document will be mostly documented within markdown files in an implementation/format known as dendron, open source and most crucially allows for tagging, and creation of very complex linking and visualization.

It is also able to seamlessly weave together maths via KaTeX (or MathJax), simple diagrams via mermaid, and at least represent code chunks. It is unable, however, to directly interface with any programming languages, and should only serve as a notes guide and literature review tool.

Note that because GitHub pages only builds websites from /docs/ occasionally, there is a bit of lag for this to update once pushed.
