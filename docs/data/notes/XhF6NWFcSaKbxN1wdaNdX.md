
This section collates some results about the importance of breaks in DFMs in the literature, and contains a small motivational study.

Note that citations currently aren't working in Dendron, but presumably this will be updated to work with pandoc. So for now, just use pandoc compatible citations (they are also easier to transfer to Rmarkdown in case).

It has been noted by many in the literature that the number of static factors has been steadily increasing.

Interestingly, it is also been reported, though less so, that the number of dynamic factors has remained quite steady. However, as there is very little work done on how the number of dynamic factors estimated can be affected by any structural breaks, it is best to be very sceptical of this result.

Thus, a compelling theory is that each break can only result in there being more factors estimated, from the perspective of a naive factor model, and if there are multiple structural breaks, then these would accumulate, thus explaining the increasing number of static factors.
