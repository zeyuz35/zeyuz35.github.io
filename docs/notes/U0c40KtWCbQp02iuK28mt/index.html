Documents summarizing estimator for qqq, the number of dynamic factors.

Note that in general, because truly dynamic factors are very difficult to estimate and work with, it is often the case that these methods will rely on a consistent estimator for the static factors rrr instead. This has the notable advantages of avoiding likelihoods, Kalman filters, etc.

Truly dynamic factor models tend to be very underdeveloped in general. Stock and Watson note in their more recent handbook chapter that there does not exist a comparison between these methods.
