Conventional Bayesian priors shrink estimates of a parameter towards its prior mean Subspsace shrinkage priors are priors on function spaces Shrink towards a class of functions E.g. Bayesian non parametric model can shrink towards linearity Here, we propose a new subspace shrinkage prior for the VAR which shrinks towards a factor model This prior is conjugate and easy to implement Scales very well to high dimensions
