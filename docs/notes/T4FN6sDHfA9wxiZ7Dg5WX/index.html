Dynamic Factor Models

Nomenclature

This thesis will refer to these as dynamic factor models, as opposed to static factor models. Confusingly, they are also referred to by [@forniGeneralizedDynamicFactorModel2000 (Private)] as Generalized Dynamic Factor Models.

Stock and Watson denote t a truly dynamic factor model as follows:

Xit=λi(L)′ft+ϵitX_{it} = \lambda_i (L)'f_t + \epsilon_{it}Xit​=λi​(L)′ft​+ϵit​

Recall that the principal components estimate is formulated to estimate the static factors, which are written as

For more details on the PC estimator, see Static Factor Models and its related sections.

Estimating the dynamic factors directly has proven to be difficult, and is done mainly via dynamic principal components. Static principal components finds the closest approximation of the covariance matrix of XtX_tXt​, among all covariance matrices of reduced rank. In contrast, dynamic principal components finds the closest approximation to the spectrum of XtX_tXt​, among all possible spectral density matrices of reduced rank.

Originally, Brillinger (1981)'s estimation algorithm generalizes static PCA to the frequency domain. The spectral dentisy of XtX_tXt​ is first estimated via some consistent spectral density estimator S^XX(ω)\widehat{S}_{XX}(\omega)SXX​(ω), at a given frequency ω\omegaω. Next, the eigenvectors corresponding to the qqq largest eigenvalues of this matrix are computed. Then, an inverse Fourier Trasform of these eigenvectors yields estimators of the principal component time series using the formulas given.

Note that because this was originally developed for classical factor analysis, the asymptotics of this under a large NNN dimension were not developed by the original author, though as expected asymptotic normality results hold.

Subsequently, [@forniGeneralizedDynamicFactorModel2000 (Private)] and [@forni (Private) 2004] study the properties of this algorithm, and establish inferential results for the resulting common component estimator. Dyanmic PCA is shown to provide piecewise consistent estimation of the component as NNN, TTT increase, and in their followup, this consistency holds if N,T→∞N, T \rightarrow \inftyN,T→∞ and N/T→0N/T \rightarrow 0N/T→0, suggesting that some caution to be exercised in finite samples if NNN is large relative to TTT.

Side note: Ben is not sure how often people actually estimate factor models via this method. Trudging through the literature, it seems that this strand was very exciting for a while, but either no one understood it well enough, or it had other significant limitations that no one actually uses it in applied research. In either case, it does not seem like a viable future path.
