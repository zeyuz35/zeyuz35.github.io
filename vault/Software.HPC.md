Entire section dedicated to setting up HPC and how to interface with it.

HPC typically has slower cores, but a very large number of threads. This makes it well suited for applications such as Monte Carlo Simulations.

## Monarch

access the filesystem directly with ssh:// login details, works best with linux file browsers.

Other OSs are really finicky and just through hissy fits.

KDE's Dolphin is also finicky with this for some reason.

## Batch Jobs

Monarch has a max job limit of 4999, not 5000 as the docs suggest.

## Linux

The HPC system runs a Linux session for you, once you tunnel in via ssh.
