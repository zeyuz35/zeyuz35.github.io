Some notes for MATLAB.

MATLAB is trash, try to convert everything to something close such as R.

MATLAB's subsetting of arrays is completing unintuitive, it tends to tranpose things when it doesn't need to, etc.

There exist R packages which have ported over most of MATLAB's functions already.

Luke Hartigan's code also includes some ported MATLAB functions.
