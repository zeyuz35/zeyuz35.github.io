Evaluation of Machine Learning in Empirical Finance

This is a note section dedicated to my honours project. 

It would be nice to re-use this and rejig it to something that is relevant to my PhD, but currently it seems a bit far fetched.

Essentially, this uses [@gu_empirical_2019]'s very comprehensive financial factor dataset, and tries applying various machine learning models to do out of sample prediction.

There are many issues with this paper, and it overall does not seem to be replicable. Even their MATLAB code for the simulation exercise does not seem to be replicable. Still, the value of the dataset is there.
