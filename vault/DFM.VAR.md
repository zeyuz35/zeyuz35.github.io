## VAR Implications

[@stock_implications_2005] provide a set of implications that the DFM (both static and VAR forms) imply, in the framework of an SVAR estimation.

1. The data has a factor structure
2. The rank of the factor innovations is q, implying q dynamic factors
3. The factor structure of the idiosyncratic shocks is such that they are uncorrelated (note that this is restrictive!)


[@stock_implications_2005] provides a method to estimate the number of dynamic factors from this set of implications; however, they require that the idiosyncratic shocks are serially uncorrelated, which is unrealistic. Therefore, it is not used much in practice.
